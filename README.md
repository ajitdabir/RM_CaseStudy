# RM_Case

**Alpha Lens – RM Capability Engine**

RM_Case is a Streamlit-based demo application for Relationship Manager capability simulation. It allows users to define a firm house view, generate realistic client case studies, submit RM responses, and evaluate recommendation quality across empathy, prudence, clarity, personalization, originality, and house-view alignment.

## Features

- Firm house view configuration
- Client case simulation
- RM response submission and evaluation
- Demo mode for stable showcase
- Performance analytics
- XP / level progression
- Clean Streamlit UI for demos
